
# AURA Accessibility Simulator - Group 1

This repository contains the Group 1 work for the AURA Accessibility Simulator assignment.

## Project Structure

* `Hlib/` - Hlib's version of the project
* `Elias/` - Elias's version of the project

Each group member works in their own folder to avoid conflicts and keep the work separated.

## Assignment Goal

The goal is to recreate a working copy inspired by the provided AURA accessibility demo.
The application should simulate accessibility improvements such as visual highlighting, profile-based interface changes, and clearer content.

## Notes

This repository is used for group work, but each student keeps their own implementation inside their personal folder.
